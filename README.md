# Hangman Game
This is my version of the classic game.

![Game Screenshot](https://github.com/rftorres/hangman-game/blob/master/screenshot/hangman.gif?raw=true)

## Requirements
- Python 3.6

  Download: `https://www.python.org/downloads`

## Usage
After you clone this repository to your desktop, go to its directory and run:

- For Windows

  `hangman-game.bat`

- For Linux

  Using terminal: `python3 hangman-game.py`
